Курсовая

Описание курсовой работы

Курсовая работа представляет собой адаптивный многостраничный веб-сайт, выполненный с использованием чистых HTML и CSS. Основной целью проекта является закрепление навыков верстки, проектирования структуры сайта и оформления пользовательского интерфейса.

В рамках курсовой реализованы обязательные элементы сайта: шапка, навигационное меню, основная часть контента, подвал и собственный логотип, выполненный средствами CSS без использования графических редакторов.

Проект может быть расширен и использован в дальнейшем для изучения серверного программирования, работы с базами данных и динамического контента.
